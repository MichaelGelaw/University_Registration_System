## Description

The University Registration System is a desktop application built using Python and Tkinter. It provides separate portals for students and institutional administrators to manage academic operations.

### Features

* Role-based access control for students and administrators.
* Student Portal: Allows students to browse the course catalog, enroll in courses, and view their transcripts.
* Admin Portal: Allows administrators to manage course offerings, student records, and grading.
